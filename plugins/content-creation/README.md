# Content Creation Plugin

AI-powered content creation tools for ArcBlock team.

## Skills

### interview-writer

AI 采访式内容创作系统。不是人写，也不是 AI 自动写，而是 AI 分析后采访人再按人的风格写。

**核心特点**：
- 通过问答沉淀用户画像（观点、写作风格、思考逻辑）
- 持续迭代演进，越用越精准
- 支持博客、社交媒体、观点文章等场景
- 一次性输出多格式、多语种

**使用方式**：
```
/interview-writer
```

或直接提供内容触发（文章链接、话题、产品发布等），系统会自动进入采访式写作流程。
