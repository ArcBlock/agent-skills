# devflow

Developer workflow automation for code review, pull requests, and daily engineering tasks.

## Installation

```bash
/plugin install devflow@arcblock-agent-skills
```

## Features

### Commands

- Code review workflows
- Pull request creation and management
- Git workflow automation

### Skills

- **code-review**: Automated code review with team standards and best practices
- **pull-request**: Create well-structured PRs with proper descriptions and context

### Agents

- Review agents for specialized analysis (security, performance, architecture)

### Hooks

- Pre-commit validation
- PR template enforcement

## Usage

```bash
# Code review
"Review the changes in this branch"

# Create pull request
"Create a PR for this feature branch"
```
