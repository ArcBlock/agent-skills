# AIStro

## 一句话定位

**AI 驱动的占星与星座应用，面向消费者的垂直 AI 产品**

---

## 产品状态

| 项目 | 状态 |
|------|------|
| **状态** | 运营中 |
| **商业模式** | 免费 + 订阅 |
| **Agent Fleet** | 计划迁移到 Agent Fleet 上运行 |

---

## 产品类型

- 消费级 AI 应用
- 基于 LLM 的个性化星座解读
- Blocklet 生态内的垂直应用

---

## 与 ArcBlock 生态的关系

| 组件 | 关系 |
|------|------|
| **Blocklet Server** | AIStro 作为 Blocklet 部署运行 |
| **AIGNE Framework** | 底层 AI 能力基于 AIGNE |
| **DID** | 用户身份管理 |
| **Agent Fleet** | **将运行在 Agent Fleet 上**，提供个性化 AI 占星 Agent |

---

## 战略意义

- 消费级产品验证 AI-Native 能力
- Agent Fleet 的首批用例之一
- Budget-Based Billing 的潜在落地场景
