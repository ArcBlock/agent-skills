## Quick Install

This will install all ArcBlock skills (like `/arcblock-commit`, `/arcblock-blocklet-dev-server`, etc.) to your Claude Code environment.

1. Run Claude Code
2. Tell Claude:

```
Ensure the target directory exists: `mkdir -p ~/.claude/simple-skills-manager-repos/arcblock`, Clone https://github.com/ArcBlock/agent-skills.git to ~/.claude/simple-skills-manager-repos/arcblock/ and execute the simple-skills-manager skill inside it, then update arcblock skills group
```
